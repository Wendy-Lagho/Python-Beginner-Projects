## Number Guessing Game (Computer)

This is a number guessing game where the computer tries to guess the number you are 
thinking of within a given range. You provide feedback to the computer whether its guess is "too high" 
or "too low" until it finds the correct number.

### How to Use
1. **Clone the repository**
```
git clone https://github.com/Wendy-Lagho/Python-Beginner-Projects.git
cd beginner-python-projects/Guess the Number (computer)
```
2. **Run the game**
```
main.py
```
